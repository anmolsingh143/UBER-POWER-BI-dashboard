🚖 Uber Data Analytics Dashboard | Power BI Project
📌 Project Overview

This project is an interactive Uber Data Analytics Dashboard built using Power BI to analyze ride performance, revenue trends, customer behavior, vehicle contribution, and operational efficiency.

The dashboard converts raw ride data into meaningful business insights to support data-driven decision-making.

🎯 Business Problem

Ride-sharing platforms generate massive volumes of data daily. However, without proper analytics:

Revenue trends remain unclear

Cancellation reasons are not fully understood

High-performing vehicle categories are unidentified

Peak demand locations and time slots are not optimized

Customer retention patterns are unknown

This project solves these problems through structured dashboard reporting.
